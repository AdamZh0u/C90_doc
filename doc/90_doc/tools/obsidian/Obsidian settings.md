# Tricks

```ad-note
title:快捷键 *推荐*

- 将表格做成快捷键 ~~计划中...~~
- 调出调试 `ctrl+shift+i`
- 界面
	- 新建笔记：`ctrl + n`
	- alt + ` 左侧边栏
	- alt + 3 右侧边栏 
	- alt + 12 前后
	- 切换编辑/预览模式：`alt + e`
	- 打开设置：`alt + ,`
	- workspace `alt + s`
	- weekntoe `alt + w`
- 文本编辑
	- 粗体：`ctrl + b`
	- **斜体（绿色上标高亮）**：`ctrl + i`
	- **添加删除线（黄色上标高亮）**：`ctrl  + d`
	- 注释：`ctrl + /`
	- 添加内部链接：`ctrl + k`
	- 添加标签：`ctrl + t`
	- 待办事项：`ctrl + enter`
	- 插入 url 链接：`ctrl + l ` 或者选中文本按住 `ctrl`不放再按 `l` 再按 `v`就可以将从浏览器中复制的 url 粘贴到此处、
	- 在当前笔记中查找：`ctrl + f`
	- Ctrl + S 搜索库
- 调用
	- **添加 admonition-mypurple** : `ctrl + shift + a`
	- 代码块：`ctrl + shift + c`，按三次可添加代码块，默认是单行代码
	-  打开命令面板：`ctrl +shift + p`
	- 调出 emoji 菜单：`ctrl + shift + e `

- **创建 ZK 卡片**：`ctrl + shift + z`

```
